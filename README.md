# SinarDigital_BE_MiniProject1_Christian_Richard_TPM37

A simple Express.js web application for submitting and viewing canteen pre orders. Built as Mini Project 1 for TPM Back End. Includes routing, form handling, controllers, middleware, and JSON data storage.

## How to Run

1. Install dependencies using:

```
npm install
```

2. Create a `.env` file based on `.env.example`.

3. Run the project:

```
npm run dev
```

4. Open http://localhost:3000 in your browser.

## Pages

- Home
- Order Form
- Orders List

## API Endpoints

- POST `/api/orders`
- GET `/api/orders`

## Tech Stack

Node.js, Express.js, dotenv, JSON
